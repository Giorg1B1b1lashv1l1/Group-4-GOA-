Introduction to JSON
JSON stand for Javascript Object Notation(ჯავასკრიფტის ობიექტის აღნიშვნა) it makes easy to share data beetween server and client-
JSON- ძალიან ჰგავს Javascript-ის ობიექტს ვიზუალურადაც და ფუნქციონალურადაც

Javascript Object--------------------------------------

cosnt myObj = {
    name:"Giorgi",
    surname:"Bibilashvili",
    age:21,
};

JSON Object-----------------------------------------------

const myObj = {
    "name":"Giorgi",
    "surname":"Bibilashvili",
    "age":21
}


ახლა როგორ გამოიყენება პრაქტიკაში , JSON -ს აქვს ასეთი მეთოდი Stringify - რომელიც ობიექტს გარდაქმნის სტრინგად , მოგეხსენებათ რომ ობიექტს არ აქვს ფიქსირებული ზომა და ის შეიძლება იყოს ძალიან დიდი მოცულობის  , ეს მეთოდი გვეხმარება ობიექტის სტრინგად გარდაქმნაში რაც ამცირებს მის ზომას,ვიცით რომ სტრინგს აქვს ფიქსირებული ზომა და ის ბევრად პატარაა ზომისაა ობიექტთან შედარებით, ლოგიკურია რომ სტრინგის ტრანსფერირება უფრო მარტივი და სწრაფია სერვერს და კლიენტს შორის ვიდრე დიდი ზომის ობიექტების სწორედ ამაში გვეხმარება JSON.

კარგით სტრინგის სახით გვაქვს ეს მონაცემები წამოღებული როგორც JSON-ის ობიექტი შეგვიძლია მას API-იც ვუწოდოთ , მაგრამ როგორ უნდა გამოვიყენოთ? ეს მონაცემები როგორ უნდა წამოვიღოთ სტრინგიდან? ამისთვის არსებობს ასეთი მეთოდი parse რომელიც გსტრინგებულ json-ის ობიექტს ისევ გარდაქმნის ჩვეულებრივ ობიექტად 


რა არის AJAX-asynchronous JavaScript and XML(ასინქრონული ჯავასკრიფტი და Extensible Markup Language(XML))
